Pair Deep Image Viewer Demos

API Documentation
Simple Interstitial Display a model in a viewer, usually used to mask the processing time of the deep image. Interstitial with Custom Loading Progress Display the interstitial viewer, but shows how to hook up your own loading indicator rather than use the default built-in animation. Simple Deep Image Viewer Display the deep image viewer using the built-in test scene and add a cube to move around. Mouse/touch to move the cube, mouse+shift to rotate the model. Deep Image Viewer with Model Display the deep image viewer using the built-in test scene and add a model to move around. Deep Image Viewer with Model and Shadow Display the deep image viewer using the built-in test scene and add a model, with floor shadow, to move around. Deep Image Viewer with Joypad Model Movement Display the deep image viewer using the built-in test scene, adds a model, and demonstrates how to move a model around using non-mouse/touch input. Deep Image Viewer with Default Swatch UI Display the deep image viewer using the built-in test scene and adds a model with color swatch options. Deep Image Viewer with Custom Swatch UI Display the deep image viewer using the built-in test scene, adds a model with color swatch options, and demonstrates how to create your own swatch UI. Deep Image Viewer with Live Data Requires API Key: Allow the user to upload an image and display the deep image viewer using the results computer from the Pair Deep Image server.
